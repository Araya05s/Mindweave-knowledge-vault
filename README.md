# Mindweave-knowledge-vault
A node-based knowledge vault inspired by draw.io, built without heavy frontend frameworks.

## Overview
This project was developed as part of my learning journey in learning Full-Stack with Laravel 12, Bootstrap, interact.JS with postgreSQL database.

The goal of this project was to explore:
- PostgreSQL database usage in Laravel 12
- Routing fundamentals
- Security with OAuth from Laravel:Breeze
- Basic architecture structure

## Tech Stack
- Laravel 12
- HTML with CSS
- PHP
- Bootstrap
- Interact.JS
